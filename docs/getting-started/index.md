# Getting Started

Welcome to Duniya! This section walks you through everything you need to
get from zero to a fully operational pharmacy in under an hour.

## Where to start

| Page | Description |
|---|---|
| **[Quickstart](quickstart.md)** | The fastest path — sign up, create a pharmacy, add products, invite a team member. |
| **[Core Concepts](concepts.md)** | The mental model — Owner, Pharmacy, Outlet, Product, Batch, and the VMI cycle. |

## Recommended reading order

1. **[Quickstart](quickstart.md)** — follow the 7-step guide end-to-end.
2. **[Core Concepts](concepts.md)** — understand the hierarchy and the VMI cycle before diving into individual modules.
3. Pick the module you use most — start with [Store Inventory](../modules/store-inventory.md).

## New here?

If this is your first time using Duniya:

- Start with the **[Quickstart](quickstart.md)**. You will be done in under 5 minutes.
- Skim **[Core Concepts](concepts.md)** to understand the terminology.
- Bookmark the **[Modules](../modules/store-inventory.md)** section — you will reference it often.

## Migrating from another system?

- Use the **[Bulk Import](../workflows/bulk-import.md)** workflow to upload your existing product list.
- Download the official template first — it ensures every field is recognised.
- See the **[Multi-Pharmacy Setup](../workflows/multi-pharmacy-setup.md)** workflow for onboarding a network of pharmacies.

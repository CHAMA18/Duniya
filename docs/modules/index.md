## How to use this reference

Each module page follows the same structure:

1. **A short intro** explaining what the module is for and why it exists.
2. **Page anatomy** — the layout and regions of the screen.
3. **Every field and action** documented in tables.
4. **Step-by-step guides** for common tasks.
5. **Tips and warnings** in callout boxes.
6. **Frequently asked questions** in collapsible sections.

Use the search bar (top right) to find any term across all modules. Use
the sidebar to browse modules by category.
